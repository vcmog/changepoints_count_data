# changepoints_count_data
Implementation of Pelt with a Poisson cost function to detect changepoints in count data over time.

Uses Ruptures to implement Pelt:
C. Truong, L. Oudre, N. Vayatis. Selective review of offline change point detection methods. Signal Processing, 167:107299, 2020.

Custom Cost class uses the Poisson cost function in the same publication.
